# 初心  
愿每个人的隐私都被保护  
但请勿用于非法用途  

# 介绍  
本软件实现文本加密，聊天加密，佛语加密  
1.加密的内容第二天自动失效，无法解密  
2.支持划词解密功能  
3.只有秘钥，才能解开

# 使用

右上角下载即可，目前只支持 windows

首先生成一个秘钥，接下来就开始愉快的聊天吧！

下载：https://github.com/futureforno/encrypt-/raw/main/%E6%96%87%E6%9C%AC%E5%8A%A0%E5%AF%86%E5%99%A8.exe
