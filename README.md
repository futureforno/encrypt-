# 初心  
愿每个人的隐私都被保护  
但请勿用于非法用途  

# 介绍  
本软件实现文本加密，聊天加密，佛语加密  
1.加密的内容第二天自动失效，无法解密  
2.支持划词解密功能  
3.只有秘钥，才能解开
